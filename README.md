# 🌤️ My Weather App

A modern weather application that provides real-time weather information for cities around the world. Built using HTML, CSS, and JavaScript, this project demonstrates API integration, asynchronous data fetching, and responsive web design.

## 🚀 Features

* Search weather by city name
* Real-time weather data
* Current temperature display
* Weather condition description
* Humidity information
* Wind speed details
* Responsive design for desktop and mobile devices
* Clean and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* Weather API (OpenWeatherMap or equivalent)

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/salma560/My-Weather-App.git
```

2. Navigate to the project folder:

```bash
cd My-Weather-App
```

3. Open `index.html` in your browser.

## 🔑 API Configuration

If the project uses a weather API:

1. Create an account on the API provider website.
2. Generate an API key.
3. Replace the placeholder API key in `script.js` with your own key.

Example:

```javascript
const apiKey = "YOUR_API_KEY";
```

## 📱 Usage

1. Enter a city name in the search box.
2. Click the search button.
3. View the current weather information for the selected city.
